# Drift Above

A dependency-free, full-screen dark-mode nature animation built with HTML, CSS, and JavaScript. It runs a 2 minute 30 second aerial flight loop with drifting terrain, glowing polygons, and occasional celebration bursts.

## Why it exists

Drift Above began as a prompt asking for a small, calming place to help lift the creator's spirits. GPT-5.6 Terra was used to turn that idea into this gentle, full-screen experience: an aerial view that drifts above the landscape, pairs encouraging messages with ambient sound, and leaves room for a little joy.

The project is intentionally lightweight and self-contained, so it can be opened whenever a quiet visual reset is welcome.

## Live site

Visit **Drift Above** at https://gregpuzzles1.github.io/drift_above/

## Running it

Open [index.html](index.html) in a modern browser to begin. Use **Pause**, **Sound**, and **A Little Joy** in the lower-right controls. Mouse movement gently offsets the camera view.

No build step or external packages are required. For viewing on other devices on the same network, serve the folder with any static web server and open its local network address in a browser.
